biggest project up to date 📈

So the biggest thing I've worked on so far was actually this issue tracking tool I called Agile Ninja.

It was totally inspired by that Jira software - you know the one most companies use for project management.

I built the whole thing using the Spring Boot framework for Java in the backend. That hooked into a MySQL database to store all the issues and comments and whatnot.

On the frontend, I leaned heavily on Bootstrap and jQuery to get the styling and interactivity worked out.

It's functional for basic tracking of issues and tasks and filtering. Right now you can create projects, add issues, assign people, that sort of thing.

But I've got some ideas for expanding it with some new features soon. Maybe add some kanban board views, Gantt chart, a calendar, custom workflows - you know, really take it to the next level of features.
